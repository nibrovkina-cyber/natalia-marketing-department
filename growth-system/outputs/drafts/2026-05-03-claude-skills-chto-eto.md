# Claude Skills что это: фича Anthropic которая заменяет промпт-инжиниринг

Anthropic выкатил Claude Skills в октябре 2025. Большинство до сих пор не понимают что это.

Я разобралась за неделю и собрала 21 готовый Skill для маркетинга малого бизнеса. Open-source, MIT, бесплатно.

Skill — это файл `SKILL.md` с инструкцией. Кладёшь в папку `~/.claude/skills/`. Claude видит и применяет.

Один скилл = одна задача. Не нужно копировать промпты из Telegram-каналов. Не нужно объяснять контекст заново. Запускаешь команду — Claude знает что делать.

Покажу что внутри `SKILL.md`, как Claude автоматически выбирает нужный скилл, и почему это убивает рынок промпт-каналов на Patreon.

---

## Что такое Claude Skills простыми словами

Skill — это **папка с одним файлом инструкции**. Внутри — название скилла, когда применять, что делать пошагово.

Claude читает все скиллы в папке `~/.claude/skills/` при старте сессии. Когда ты пишешь команду или задачу — Claude сам выбирает подходящий скилл и применяет его.

Аналог: у программиста есть библиотеки. Ты не пишешь сортировку с нуля — ставишь библиотеку и вызываешь функцию.

С Skills так же. Ты не объясняешь Claude как писать продающий текст с нуля — ставишь скилл `/natalia-copy` и вызываешь.

## Чем Skills отличаются от обычных промптов

| Параметр | Промпт из Telegram | Claude Skill |
|---|---|---|
| Где живёт | в чате, теряется | в папке, постоянно |
| Контекст | каждый раз заново | сохраняется |
| Запуск | копировать-вставить | команда `/skill-name` |
| Обновления | руками | через git pull |
| Делиться | отдельные файлы | целый репо |
| Поиск | "где я его сохранила" | автоматически |

Промпт — одноразовый рецепт. Skill — это блюдо в меню ресторана.

## Структура файла SKILL.md изнутри

Минимальный Skill выглядит так:

```markdown
---
name: my-first-skill
description: Что делает этот скилл и когда его применять
---

# Инструкция

Когда пользователь просит [задачу]:
1. Шаг первый
2. Шаг второй
3. Финальный output

## Правила
- Что нельзя делать
- Какой формат соблюдать
```

`name` — короткое имя.
`description` — самая важная часть. Claude читает её, чтобы понять когда скилл подходит.

Если description чёткий — Claude применит скилл когда нужно. Если размытый — будет путаться или игнорировать.

---

## Как Claude автоматически выбирает нужный скилл

Когда ты пишешь команду или задачу, Claude:

1. Читает description всех Skills в папке
2. Сравнивает с твоей задачей
3. Выбирает подходящий
4. Применяет инструкцию

Ты можешь и явно указать: `/natalia-copy` запускает копирайтера сразу, без подбора.

Если скиллов 21, как в natalia-marketing-department, оркестратор `/natalia` знает обо всех 21 и выбирает нужный под задачу. "Сделай мне план запуска продукта" → запускается `/natalia-launch`. "Перепиши лендинг" → `/natalia-landing`.

## Как написать свой первый Skill за 10 минут

Шаг 1 — создай папку:
```bash
mkdir -p ~/.claude/skills/my-first-skill
cd ~/.claude/skills/my-first-skill
```

Шаг 2 — создай файл `SKILL.md`:
```markdown
---
name: my-first-skill
description: Помогает написать пост в Telegram для эксперта-консультанта. Применяй когда пользователь просит "напиши пост для Telegram"
---

# Инструкция

Когда пользователь просит пост для Telegram:

1. Спроси тему поста
2. Напиши хук в первой строке (вопрос или конкретный факт)
3. 3-5 коротких параграфов раскрытия
4. CTA в последней строке
5. 2-3 хэштега

## Правила
- Параграф 1-2 предложения
- На "ты", не на "вы"
- Без эмодзи в начале
- Без воды
```

Шаг 3 — открой Claude Code и попроси: "Напиши пост для Telegram про утренние ритуалы"

Claude автоматически применит твой скилл.

## FAQ

**Чем Claude Skills отличаются от ChatGPT custom GPTs?**
Custom GPT — закрытый конструктор внутри платформы OpenAI. Claude Skill — открытый файл `.md`, который ты редактируешь руками, версионируешь в git, делишься через GitHub.

**Можно ли использовать Skills бесплатно?**
Сами Skills — бесплатно. Платишь только Anthropic API за токены. $5-30 в месяц на средний объём задач.

**Где скачивать готовые Skills?**
GitHub. Например, natalia-marketing-department — 21 Skill для маркетинга. Поиск по тегу `claude-skills` находит сотни.

**Можно ли продавать собственные Skills?**
Да. Skill — это твой код. Лицензируй как хочешь. Многие продают платные коллекции на Gumroad.

**Работают ли Skills в Claude Code и Claude Desktop одновременно?**
Да. Папка `~/.claude/skills/` общая для обоих клиентов. Один раз поставил — работает везде.
