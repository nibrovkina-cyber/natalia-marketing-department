## FAQ — 5 вопросов

**1. Сколько стоит работа Claude для маркетинга?**
$5-30 в месяц на Anthropic API в зависимости от объёма задач. Сами 21 агент бесплатно.

**2. Нужно ли уметь программировать?**
Нет. Достаточно установить Claude Code (5 минут) и научиться запускать команды. Это не написание кода.

**3. Можно ли использовать в России?**
Да. Anthropic API работает в РФ через прямую оплату или зарубежные карты. Сами скиллы локальные.

**4. Чем отличается от ChatGPT с готовыми промптами?**
ChatGPT — универсальный AI без методологии. natalia — 21 узкий агент на конкретной школе direct response (Schwartz/Ogilvy/Hopkins). Каждый делает одну задачу хорошо.

**5. Кому это НЕ подойдёт?**
Крупным агентствам с >50 человек, корпорациям с юридическими ограничениями, тем кто хочет "одну кнопку" без редактуры. Это инструмент мастера, не магия.
