# Как сделать маркетинг без маркетолога: пошаговая система для малого бизнеса

Маркетолог в Москве стоит 100-200 тысяч в месяц. Агентство — от 150 до 500. У малого бизнеса нет таких денег.

При этом маркетинг нужен. Без него — нет заявок, нет роста, конкуренты обгоняют.

Я пять лет работала маркетологом. Потом увидела как 90% задач закрывают AI-агенты на Claude. Собрала 21 готовый скилл в open-source — позиционирование, копирайтинг, лендинг, email, реклама, SEO. Бесплатно, MIT.

В этой статье — пошаговая система как закрыть работу маркетингового отдела одним человеком + Claude. Реальные кейсы стоматологии и кафе. Что делать в первую неделю и какой результат ждать через месяц.

---

## Почему маркетолог 100К — устаревшая модель

В классическом виде маркетолог делает:
- Стратегия и позиционирование (1 раз в год)
- Копирайтинг (постоянно)
- SMM (постоянно)
- SEO-статьи (по плану)
- Реклама (настройка + ведение)
- Email-рассылки
- Анализ конкурентов

Это разные навыки. Один человек редко владеет всеми хорошо.

Что меняется с AI:
- Стратегия → промпт + методология (Schwartz/Ogilvy/Hopkins)
- Копирайтинг → скилл `/natalia-copy` за 5 минут вместо 3 часов
- SMM → контент-план + атомизатор
- SEO → структура + draft за 1 час вместо дня
- Реклама → тексты объявлений за минуты
- Email → готовая welcome-серия за час

Маркетолог становится оператором системы, не исполнителем.

### Технический минимум
- Claude Code (бесплатно)
- Anthropic API ключ ($5-30/мес)
- 5-10 часов в неделю

### Методологический минимум
Прочитать в общих чертах:
- Schwartz "Breakthrough Advertising" — 5 уровней осознанности
- Ogilvy "Confessions of an Advertising Man" — заголовки
- Hopkins "Scientific Advertising" — verifiable claims

Не нужно читать целиком. Достаточно знать что внутри методологии — Claude применяет за тебя.

### Файловый минимум
- Папка клиента с brand-memory (адрес, цены, ICP, что говорить и что запрещено)
- 21 Skill из natalia-marketing-department в `~/.claude/skills/`

## Первая неделя: с чего начать

### День 1 — установка и позиционирование (2 часа)

Ставишь Claude Code. Клонируешь репо:

```bash
git clone https://github.com/nibrovkina-cyber/natalia-marketing-department.git
mkdir -p ~/.claude/skills
cp -r skills/* ~/.claude/skills/
```

Создаёшь папку клиента с файлом `brand-memory.json` — туда пишешь адрес, цены, ICP, что говорить и что запрещено.

Запускаешь:
```
/natalia-positioning
```

Через 15 минут получаешь стратегический бриф — точку отстройки от конкурентов, целевую аудиторию по уровням Швартца, три ключевых сообщения.

### День 2 — разведка конкурентов (1 час)

```
/natalia-competitors
```

Claude находит 3-5 конкурентов в твоей нише, разбирает по 7 пунктам: позиционирование, цены, аудитория, слабости, сильные стороны, контент, конверсия.

### День 3 — лендинг (3 часа)

```
/natalia-landing https://your-current-site.ru
```

Аудит текущей страницы. Указывает что переписать. Даёт варианты переписки на основе позиционирования из дня 1.

### День 4 — лид-магнит (2 часа)

```
/natalia-leadmagnet
```

Claude предлагает 3 варианта лид-магнита под твою аудиторию + полную сборку выбранного варианта (PDF / чек-лист / шаблон).

### День 5 — контент-план (1 час)

```
/natalia-social
```

Месячный контент-план для Telegram и/или VK. С темами, форматами, графиком публикаций.

### День 6-7 — анализ и план

Смотришь что получилось. Что работает — оставляешь. Что не работает — переделываешь через те же скиллы.

## Реальный кейс: стоматология на Покровке

Сеть из двух точек на Китай-городе. Один врач, две ассистентки. Маркетолога нет.

**До:** 12 заявок в неделю. Лендинг шаблонный. Реклама в Яндексе на копировальный текст конкурентов.

**Что сделали за 6 недель:**
1. `/natalia-competitors` — нашли 3 конкурентов в радиусе 1 км. Их слабость — все говорят "качественное лечение", никто не даёт цены вперёд
2. `/natalia-positioning` — точка отстройки: "цена зафиксирована до начала лечения"
3. `/natalia-landing` — переписка первого экрана и блока цен
4. `/natalia-ads` — тексты для Яндекс Директа на эту точку отстройки
5. `/natalia-email` — welcome-серия для записавшихся

**После:** 47 заявок в неделю. Без увеличения рекламного бюджета. Без агентства.

## Реальный кейс: греческое кафе в Митино

Один владелец. SMM делал сам, выгорал. Думал нанимать SMM-щика за 30 тысяч.

**Что сделали:**
1. `/natalia-brand` — tone of voice (тёплый, по-домашнему, без корпоративности)
2. `/natalia-social` — месячный контент-план VK + Telegram
3. `/natalia-atomizer` — каждый длинный пост → 5 коротких + 3 рилса + 2 сторис

**Результат:** 4 часа в неделю на контент вместо 20. Не нанял SMM-щика — сэкономил 30К/мес.

## FAQ

**Можно ли совсем без маркетингового образования?**
Можно. Скиллы написаны так, что применяют методологию за тебя. Ты только редактируешь финальные тексты под свой голос.

**Сколько часов в неделю реально нужно?**
5-10 часов. Если делать всё руками — 30-40 часов.

**Что если бизнес сезонный?**
В межсезонье — пишешь SEO-статьи через `/natalia-seo`. В сезон — реклама и контент.

**А если у меня услуги, не товары?**
Все скиллы работают для услуг. Стоматология, коучинг, юридические услуги — кейсы есть.

**Когда уже видно результат?**
Первые заявки — 2-4 недели. Стабильный поток — 6-8 недель. Это математика SEO + контент-маркетинга, не магия.
