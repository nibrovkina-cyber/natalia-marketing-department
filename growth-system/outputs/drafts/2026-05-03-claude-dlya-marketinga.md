## Что такое Claude и почему именно он для маркетинга

### Claude vs ChatGPT для русского рынка

ChatGPT — универсальный AI. Знает всё понемногу. Без методологии.

Claude от Anthropic — другой подход. Длинный контекст (можно подавать целиком книги Огилви и брендбук клиента), точное следование инструкциям, и главное — поддержка Claude Skills.

Claude Skill — это файл `SKILL.md` с инструкцией. Один скилл = одна задача. Кладёшь в папку — Claude видит и применяет.

Для маркетинга важно: AI должен следовать методологии, а не импровизировать. Claude следует.

### Что такое Claude Code и зачем он нужен

Claude Code — это терминальный клиент Claude от Anthropic. Бесплатно. Запускается на твоём компе, работает с твоими файлами.

В отличие от веб-чата, Claude Code умеет:
- Читать папки клиентов локально
- Запускать скиллы по команде
- Сохранять brand-memory между сессиями
- Создавать файлы (лендинги, презентации, посты)

### Сколько стоит работа Claude для маркетинга

Сам Claude Code — бесплатно.
Сами 21 агент — бесплатно (MIT лицензия).

Платишь только Anthropic API за токены. На средний объём задач для одного клиента — $5-30 в месяц. Один бриф клиента — около $0.30. Один полный лендинг с разбором — $1-2.

Для сравнения: подписка ChatGPT Plus — $20/мес фиксированно.

### Методология Огилви / Швартца / Хопкинса в каждом скилле

Огилви — заголовки. Доказано: заголовок читают в пять раз больше людей чем весь остальной текст. Скилл `/natalia-copy` всегда даёт 3-5 вариантов заголовка, не один.

Швартц — пять уровней осознанности аудитории. Каждый текст пишется под конкретный уровень. Не путаешь "холодного" клиента с "горячим" — конверсия в два раза выше.

Хопкинс — verifiable claims. Никаких "лучший способ" — только конкретные доказуемые цифры. Скилл блокирует попытку Claude написать "качественное лечение" или "опытные специалисты".

### Чем эти агенты отличаются от готовых промптов

Промпты из Telegram-каналов — одноразовые. Каждый раз заново копируешь, объясняешь контекст, правишь.

Скилл — постоянный. Лежит в `~/.claude/skills/`. Запускается командой. Помнит методологию. Читает данные клиента из папки автоматически.

Это разница между списком советов и работающей системой.

## Как установить за 5 минут

### Что нужно

- Claude Code (бесплатно с claude.com)
- Anthropic API ключ (оформляется в console.anthropic.com)

API работает в РФ через прямую оплату. Если карта не проходит — есть варианты с зарубежными картами.

### Команды установки

**macOS / Linux:**

```bash
git clone https://github.com/nibrovkina-cyber/natalia-marketing-department.git
cd natalia-marketing-department
mkdir -p ~/.claude/skills
cp -r skills/* ~/.claude/skills/
```

**Windows (PowerShell):**

```powershell
git clone https://github.com/nibrovkina-cyber/natalia-marketing-department.git
$src = ".\skills\"
$dst = "$env:USERPROFILE\.claude\skills\"
New-Item -ItemType Directory -Force -Path $dst
Copy-Item -Path $src* -Destination $dst -Recurse -Force
```

### Первый запуск

Открой папку клиента в Claude Code. Напиши:

```
/natalia-positioning
```

Через 5 минут получишь стратегический бриф: угол позиционирования, точку отстройки от конкурентов, три триггера аудитории.

## Чего Claude для маркетинга НЕ делает

Claude — мощный инструмент. Но он не магия.

Не заменяет редактуру голоса бренда. AI пишет хорошо технически, но твой голос — это твой голос. Финальная правка остаётся.

Не публикует контент автоматически. Тексты надо копировать в VK / Telegram / лендинг руками.

Не работает без человека-стратега. Скиллы дают тебе результат, но решение что делать дальше — твоё.

Это инструмент для маркетолога-одиночки или владельца бизнеса. Не "одна кнопка для маркетинга".

## FAQ

**Сколько стоит работа Claude для маркетинга?**
$5-30 в месяц на Anthropic API в зависимости от объёма задач. Сами 21 агент бесплатно.

**Нужно ли уметь программировать?**
Нет. Установка — 5 минут. Запуск — одна команда. Это не написание кода.

**Можно ли использовать в России?**
Да. Anthropic API работает в РФ через прямую оплату или зарубежные карты. Скиллы локальные.

**Чем отличается от ChatGPT с готовыми промптами?**
ChatGPT — универсальный AI без методологии. natalia — 21 узкий агент на конкретной школе direct response (Schwartz/Ogilvy/Hopkins). Каждый делает одну задачу хорошо.

**Кому это НЕ подойдёт?**
Крупным агентствам с большой командой. Корпорациям с юридическими ограничениями. Тем кто хочет "одну кнопку" без редактуры. Это инструмент мастера.
