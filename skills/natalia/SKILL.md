# natalia · marketing department

16 AI-агентов для маркетинга малого и среднего бизнеса в России. Каждый агент — узкий специалист с зашитой методологией:

- **Ogilvy** — заголовки, named-audience, sell-the-effect
- **Schwartz** — 5 уровней осведомлённости аудитории
- **Hopkins** — цифры вместо прилагательных, scientific advertising

## Workflow

Запуск любого агента:
```
/natalia-positioning  Стоматология на Покровке. Цена 4500 за пломбу. Конкуренты — соседние клиники с тем же чеком.
/natalia-landing      https://medeadent.tilda.ws
/natalia-emails       welcome-серия для онлайн-курса по B2B-продажам
/natalia-ads          B2B-консалтинг для ритейла, бюджет 80 000 ₽/мес, регион — Москва
```

Каждый агент **сам спросит** недостающий контекст. После ответа возвращает обязательный блок `🧠 Почему я так сделал` с расшифровкой по Ogilvy/Schwartz/Hopkins.

## Cross-skill handoffs

Агенты подсказывают какие следующие skills логично применить:

- positioning → copy → landing → ads
- leadmagnet → emails → funnel
- competitors → positioning → ads
- launch → copy → emails → ads
