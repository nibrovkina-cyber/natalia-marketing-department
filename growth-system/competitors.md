## Косвенные конкуренты в РФ

### 1. ChatGPT Plus + промпт-сборники

**Что:** подписка $20/мес + готовые промпты которыми все обмениваются.

**Слабость:**
- Нет методологии (Schwartz/Ogilvy/Hopkins)
- Нет специализации под маркетинг
- Промпты разрозненные — не система
- Каждый раз заново объясняешь контекст

**Как позиционироваться:** "ChatGPT — это AI. natalia — это маркетинговый отдел."

### 2. GigaChat (Сбер)

**Что:** российский LLM. Бесплатный для базового.

**Слабость:**
- Хуже на креативе и копирайтинге
- Не специализирован
- Нет MCP / Skills архитектуры
- Не работает с Claude Code

**Где сильнее:** работает в РФ без VPN, оплата через российские карты.

**Как позиционироваться:** для маркетинга специально — Claude > GigaChat. natalia — это поверх Claude.

### 3. Yandex GPT (через YaCloud / Алиса Pro)

**Что:** Яндексовский LLM.

**Слабость:**
- Универсальный
- Не для маркетинга специально
- Нет инфраструктуры под Skills

**Игнорируем как прямого конкурента.**

### 4. Курсы по AI-маркетингу (Skillbox, Нетология, Skypro)

**Что:** обучают как использовать AI в маркетинге. Стоят 30-150К.

**Слабость:**
- Учат теории. Не дают **готовый инструмент**.
- 3-6 месяцев обучения. Студент в конце имеет знания, не пайплайн.
- Курс устаревает за полгода.

**Как позиционироваться:** "Курс даёт знания. natalia даёт работающую систему."

### 5. Платные SaaS для AI-маркетинга в РФ (Texti.fm, Copywriter.ai от Тинькофф, etc.)

**Что:** SaaS-инструменты для генерации текстов.

**Слабость:**
- Подписка 1500-5000 ₽/мес
- Закрытый код — не настроишь под себя
- Универсальный — не методология
- Зависит от вендора

**Как позиционироваться:** "Платный софт зависит от вендора. natalia — open-source, твой код."

### 6. Агентства AI-маркетинга в РФ (новый тренд 2025-2026)

**Что:** агентства, которые позиционируются как "делаем маркетинг через AI". Берут 100-300К/мес.

**Слабость для нашего ICP:**
- Дорого
- ICP не может себе позволить
- Внутри они часто используют те же инструменты что доступны всем

**Как позиционироваться:** "Агентство берёт 200К/мес чтобы запустить ChatGPT. С natalia ты делаешь то же сама за $5 в месяц на API."
