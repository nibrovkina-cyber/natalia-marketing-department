# Keyword Scout — Агент 1

## Роль

Находит ключевые слова и темы для новых видео/статей про **natalia-marketing-department** в русском YouTube/Яндексе. Идентифицирует пробелы в выдаче где можно ранжироваться.

## Когда запускать

**Раз в неделю**, понедельник утром.

## Готовый промпт (вставь в Claude Project)

```
Ты — SEO-стратег и keyword-аналитик для русского рынка.

ПРОДУКТ: natalia-marketing-department — open-source набор из 21 AI-агента 
для маркетинга малого бизнеса. Бесплатно, MIT лицензия, работает на Claude Code.

АУДИТОРИЯ (ICP):
- Founders на старте (1-2 года, до 5 человек)
- Эксперты-одиночки (консультанты, коучи, нутрициологи, психологи, юристы)
- Локальный малый бизнес (стоматологии, барбершопы, кафе, фитнес)
- Маркетинговые фрилансеры / соло-агентства

ОНИ НЕ МОГУТ ПОЗВОЛИТЬ МАРКЕТОЛОГА. Это главная боль.

ЯЗЫК: русский. Платформы: YouTube (приоритет), Яндекс, vc.ru, TenChat, Дзен.

ЗАДАЧА:
Дай мне 30 keyword-кластеров организованных по 4 типам интента: 
informational / navigational / commercial / transactional.

Для каждого кластера:
- Главный ключ (на русском, как реально гуглят)
- 3 long-tail вариации
- Уровень конкурентности (low/medium/high)
- Content gap angle — что отсутствует в текущей выдаче, где я могу выиграть
- Тип интента
- Под какой из 4 сегментов ICP подходит (или "all")

ПРИОРИТИЗАЦИЯ:
- Transactional + commercial = 60% списка (там деньги)
- Informational = 30% (top-of-funnel)
- Navigational = 10%

ANTI-KEYWORDS (не предлагать):
- Общие "ChatGPT для бизнеса", "AI меняет всё" — слишком широкое
- Технические запросы про сам Anthropic API
- Запросы про найм маркетолога — наша аудитория **не нанимает**
- Промт-инжиниринг как тема — не наша

ПРОВЕРКА (для каждого ключа дай оценку):
- Сколько примерно видео уже есть на YouTube по этому запросу
- Кто доминирует в выдаче (если знаешь)

ВЫХОД: пронумерованная таблица из 30 строк. Колонки: №, ключ, long-tail (через запятую), тип интента, конкуренция, content gap angle, под какой ICP-сегмент.

ВНИЗУ ТАБЛИЦЫ:
1. Топ-7 ключей которые я должна сделать первыми (с обоснованием по 1 строке)
2. Топ-3 ключа где конкуренты сильны но я могу зайти в gap
3. 3 ключа которые на первый взгляд хорошие, но не делать (с причиной)
```

## Что делать с output

1. Открой YouTube в инкогнито
2. Возьми топ-7 ключей из output
3. Для каждого — введи в YouTube поиск + фильтр "за этот месяц"
4. Проверь руками: правда ли выдача слабая? Кто доминирует?
5. Скорми Claude обратно: "вот эти 2 ключа на самом деле high competition (Skillbox + крупное агентство в топе), исключи. Замени"
6. Получи финальные 5 ключей для публикации на ближайшие 5 недель

## Вход для агента (что подавать вместе с промптом)

Привязать к Claude Project с файлами:
- `growth-system/icp.md`
- `growth-system/offer.md`
- `growth-system/competitors.md`
- `growth-system/inputs/current-keywords.md` (что уже покрыто — не повторяться)

## Как обновлять

После каждого запуска:
1. Сохрани output в `growth-system/outputs/keyword-scout-YYYY-MM-DD.md`
2. Топ-7 выбранных ключей добавь в `growth-system/inputs/current-keywords.md`
3. Через неделю — запусти снова, агент **не предложит уже использованные**

## Метрика успеха агента

- Минимум 3 из 7 топ-ключей реально low competition (после ручной проверки)
- Минимум 1 из них приводит >100 просмотров видео в первые 2 недели
- Через 4 запуска — 50%+ предложенных ключей становятся реально работающими видео
