# Video Series — "Я нагоняю трафик публично"

## Концепция

Публичный реалити-шоу: ты разворачиваешь growth-систему для natalia-marketing-department в режиме build in public. Каждое сделанное действие = новое видео.

**Двойной актив:**
- Контент сам по себе работает (трафик на репо)
- Сам процесс становится кейсом для следующих продуктов

## Целевая длительность серии

8 недель. Чтобы охватить полный цикл от запуска до первых результатов.

## Структура каждого эпизода

| Часть | Время | Что |
|---|---|---|
| Хук | 0-15 сек | Конкретный результат или провал этой недели |
| Контекст | 15-60 сек | Что было в прошлой неделе, почему это видео сейчас |
| Действие | 1-7 мин | Сама работа: показываем экран Claude Code, реальные команды |
| Цифры | последние 30-60 сек | Что вышло — просмотры/звёзды/клики/регистрации |
| CTA | финал | Ссылка на репо, подписка, "увидимся через неделю" |

## Длительность

10-15 минут на эпизод. **Не shorts.** Реалити-формат лучше работает в long-form.

Для ютуб-шортс — нарезаешь по 30-60 сек ключевых моментов из long-form.

## Формат съёмки

- Talking head + screen capture в равных пропорциях
- Голосовая дорожка дома, не студия
- Минимум монтажа — реалити-формат
- Никаких заставок длиннее 3 секунд
- Title cards между разделами

## Что готовить заранее

Перед стартом серии (раз):
- [x] Структура `growth-system/` папки
- [x] Файлы стратегии (icp/offer/voice/competitors)
- [ ] Все 6 агентов готовы в Claude Projects
- [ ] План видео #0 — анонс
- [ ] Канал-плейлист "Я нагоняю трафик публично" на YouTube
- [ ] Шаблоны для thumbnail (один стиль на всю серию)

Перед каждым эпизодом (1-2 часа):
- Подготовить демо в Claude Code (что показываем на экране)
- Тезисы (не сценарий) — 5-7 точек
- Цифры за неделю (свежие)

## Метрика успеха серии

**Через 4 эпизода:**
- 50+ комментариев суммарно
- 10+ Github звёзд от зрителей
- 5+ DM "повторил у себя, работает"

**Через 8 эпизодов:**
- 100+ Github звёзд
- 1000+ visits на репо
- 5+ kostumerов hosted-версии (если запущена)
- Минимум 1 видео в серии набирает >5 000 просмотров

## Что в каждом эпизоде должно быть обязательно

1. **Хук с конкретной цифрой** в первые 10 секунд
2. **Минимум 1 показ экрана** Claude Code в реальной работе
3. **Реальные цифры** статистики (даже если плохие)
4. **Ссылка на репо** в описании + закреплённый коммент
5. **Дата записи** в каптионе (build-in-public трейл)
