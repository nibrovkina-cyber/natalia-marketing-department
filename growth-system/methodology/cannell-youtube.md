# Sean Cannell — YouTube методология

**Источник:** Sean Cannell, CEO Think Media (~3M subs), книга "YouTube Secrets" — №1 бестселлер по YouTube-стратегии.

Две взаимосвязанные методологии: **7 Cs** (фундамент канала) + **7 Rs** (операционный цикл на видео). Применяются вместе.

---

## 7 Cs — фундамент (один раз при старте канала)

| C | Принцип | Что делать |
|---|---|---|
| **Courage** | Включить страсть, побороть страх камеры | Найти тему за которую ты готов "умирать" — без страсти зрители почувствуют фальшь |
| **Clarity** | Чёткая ниша + понятный target viewer | Один тип зрителя, одна основная тема. "Все" = никто |
| **Channel** | Визуальный бренд | Лого, баннер, описание, единый стиль обложек, единый Tone of Voice |
| **Content** | Хук в первые 15 сек + value prop + CTA | Каждое видео: первая секунда зацепляет, последняя — приглашает |
| **Consistency** | Устойчивый темп без выгорания | Лучше 1 видео в неделю год, чем 5 в неделю месяц |
| **Community** | Превращать зрителей в суперфанов | Отвечать на каждый комментарий первые 24 часа |
| **Cash** | Монетизация | AdSense + спонсоры + собственный продукт (как у тебя — natalia-marketing-department + hosted) |

## 7 Rs — операционный цикл (на каждое видео)

### R1 — Reverse Engineering
Перед съёмкой: разобрать **3-5 конкурентов** в твоей нише по 5 пунктам:
- Их топ-видео (по просмотрам)
- Структура хука первые 15 сек
- Title + thumbnail формула
- Watch time / retention где доступно
- Где content gap — что они НЕ делают

### R2 — Research
Темы и ключи **до съёмки**, не после. У нас уже есть Keyword Scout агент в [growth-system/agents/keyword-scout.md](../agents/keyword-scout.md).

### R3 — Recording
- План кадров (storyboard) перед записью
- Гир: микрофон важнее камеры
- Энергия x1.5 от обычной — на видео всё кажется тише

### R4 — Releasing
**Самое часто игнорируемое.** Один час на:
- Title — A/B тест 3 варианта через YT Studio
- Thumbnail — A/B тест 3 варианта через YT Studio
- Description — первые 150 символов = preview в search
- Теги — 5-15 релевантных
- End screen + cards — ссылки на другие свои видео
- Pinned comment с CTA + ссылками

### R5 — Rocketing
Первый час после публикации = критический сигнал алгоритму:
- Внешний трафик с TenChat, Telegram, Twitter, vc.ru, рассылки
- Уведомления подписчикам через Community tab
- Оплачиваемая реклама на других платформах (опционально)

**То что мы делаем через distribution-packs — это Rocketing.**

### R6 — Reviewing
Через 7 + 30 дней — анализ:
- CTR thumbnail (норма 4-8%, выше 10% = вирусное)
- Avg view duration (для 10-минутного видео >50% retention = алгоритм поднимает)
- Drop-off в первые 30 секунд = плохой хук
- Топ-источники трафика (suggested / search / external / browse)
- Demographics зрителей

### R7 — Repeating
Усиливать то что сработало:
- Тема дала 3x просмотров → серия из 5 видео по этой теме
- Thumbnail-формула сработала → копируешь на следующие
- Конкретный хук дал retention → используешь в новом видео

Отказываться от провалов. Cannell: "20% твоих видео дадут 80% результата. Найди эти 20% и копируй."

## Применение к каналу @NateBrovk

**Состояние на 2026-05-03:**
- 3 long-form (29 / 148 / 121 просмотров)
- 3 Shorts (16 / 866 / 400)
- Челлендж "Я нагоняю трафик публично" — Эпизоды 0, 1 в plan, 2 написан

**Где сильно:**
- ✅ Clarity: ниша AI-маркетинг для SMB чёткая
- ✅ Cash: есть продукт (open-source + hosted)
- ✅ Rocketing: запущен distribution-flow (Telegraph, TenChat, Telegram, Habr, Дзен)

**Где слабо (priority fix):**

### 🚨 Releasing (R4) — самое срочное
Видео `UrU_tapUIxM` "Как создать AI маркетинговый отдел в Claude Code (16 скиллов, бесплатно)" даёт 29 просмотров за сутки. Это либо low-CTR thumbnail, либо learning-style title.

**Что сделать:**
1. YouTube Studio → видео → **Test & Compare** для thumbnail
2. Загрузить 2-3 варианта thumbnail (например, A: эмоциональное лицо + цифра 12→47, B: до/после скриншоты, C: твоё лицо + "$5/мес")
3. YouTube тестирует автоматически 14 дней
4. Победитель остаётся

**Test для title:** в YouTube Studio есть тоже A/B title — попробуй:
- "Как создать AI маркетинговый отдел в Claude Code" (current)
- "Я заменила маркетолога на AI за $5/мес. Стоматология выросла в 4 раза"
- "21 AI-агент для маркетинга малого бизнеса. Бесплатно, open-source"

### Reviewing (R6) — через 7 дней
Зайти в YT Studio Analytics → выписать в [traffic-data.csv](../inputs/traffic-data.csv):
- CTR каждого видео
- Avg view duration
- Drop-off график
- Топ источников

### Channel (C3) — низкоприоритетно
- Баннер канала (если ещё default или слабый)
- Единый стиль обложек серии "Я нагоняю трафик публично" (например, цифра эпизода в углу + единая палитра navy/gold)
- Описание канала: "Каждую среду — новый эпизод челленджа от 0 до 10K за 8 недель"

## Cannell vs Greg Isenberg — два бенчмарка

| Аспект | Cannell | Isenberg |
|---|---|---|
| Платформа | YouTube главная | Twitter + YouTube |
| Метрика успеха | Watch time + retention | Engagement + outbound traffic |
| Подход к контенту | Перфект-видео-рецепт + A/B | Build in public + быстрая итерация |
| Применимость для тебя | На 80% (твой основной канал = YouTube) | На 30% (но "build in public" формат — 100%) |

**Совмещаем:**
- Cannell-методика **на каждое видео отдельно** (структура + Releasing + Reviewing)
- Isenberg-формат **на всю серию** (build in public, прозрачные цифры)
