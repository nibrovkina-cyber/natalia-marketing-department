## Принципы (из плейбука)

1. **Build in public** — каждое движение системы публично. Провалы тоже.
2. **Один агент — одна работа** — не делать супер-агента.
3. **voice.md побеждает** — все агенты сверяются перед output.
4. **Iteration > Perfection** — публикуй среднее быстро, чем идеальное никогда.
5. **Цифры > прилагательные** — везде в материалах.
6. **Дистрибьюция > контент** — статья 7/10 + сильная дистрибьюция > статья 10/10 без неё.
7. **SEO даёт результат с недели 4-6** — не бросай раньше.
